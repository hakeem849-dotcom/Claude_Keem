# PharmaDesk Personal Use Only

This branch/route is for a separate personal-use-only version of PharmaDesk.

Guardrails:
- Do not enter real patient names, DOBs, addresses, Rx numbers, insurance IDs, claim IDs, prescriber names, phone numbers, or other PHI.
- Use fictional or fully de-identified placeholder entries only.
- This is a workflow aid and learning sandbox, not a clinical record, DUR engine, pharmacy management system, controlled-substance log, billing system, or source of truth.
- Confirm all clinical, dispensing, payer, inventory, counseling, PMP, and documentation actions in the approved pharmacy system.
- Do not rely on this app for patient-care decisions.

Branch intent:
- Keep personal-use experiments isolated from the main public demo app.
- Build features around non-PHI task triage, reminders, checklists, SOP prompts, and personal workflow reflection.
